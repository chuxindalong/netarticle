加油猫带给你元气.

猫猫的原型是公司一直名叫炸炸的橘猫，因此漫画里的炸炸也常常穿着黄白相间和服。干巴爹是由日语音译为汉语的，粉色的扇子是日本漫画中常见的元素。小爱心点缀显得画面更加活泼。

设计想法

恋爱猫向你表白.

阿姨洗铁路，这幅是经典的日语“我爱你”在中文里的读音。这里的炸炸眼睛弯弯的，双手比作心形，脸蛋更圆一些。身后还有一颗大大的爱心。整幅画面都很圆润，充满爱心元素。

  • 设计师: Rainie/宗悦
  • 分类: 日常
